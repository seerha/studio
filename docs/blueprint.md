# **App Name**: GovBook Portal

## Core Features:

- User Registration & Verification: Dual OTP verification and document upload for user categorization (Govt., NGO, Private).
- Event Proposal Submission: Allow verified users to submit event details, including VIP status and security clearance documents. Dates are tentatively blocked upon submission.
- Admin Approval Workflow: A multi-tier approval queue for verifying officers and approving authorities, ensuring compliance with government guidelines.
- Automated Tariff Calculation: Automatically calculates rent and security deposit based on the user's category and event details.
- Secure Payment Gateway Integration: Integrate with the State/Govt Payment Gateway for secure transaction processing with payment link.
- Booking Allotment Letter Generation: Generate a secure PDF Booking Allotment Letter with a unique Booking Reference ID upon successful payment.
- Post-Event Clearance and Refund: Enable Auditorium Manager to submit clearance reports and initiate automatic security deposit refunds.

## Style Guidelines:

- Primary color: Deep blue (#003366) to convey authority and trust, reminiscent of government institutions.
- Background color: Light gray (#F2F2F2) for a clean and professional appearance.
- Accent color: Gold (#FFD700) to highlight important information and CTAs, adding a touch of prestige.
- Body and headline font: 'Inter' sans-serif font for clear readability and a modern, neutral appearance.
- Note: currently only Google Fonts are supported.
- Use clear and professional icons to represent different categories, actions, and statuses. Preferably the open source 'iconic' set
- Maintain a clean and organized layout with clear visual hierarchy and intuitive navigation, ensuring accessibility and usability.
- Incorporate subtle animations and transitions to provide feedback and enhance user experience, without being distracting.
